# 32-Bit Single-Cycle Processor in Verilog

## Overview
This project implements a 32-bit Single-Cycle Processor using Verilog HDL and Xilinx Vivado.

## Modules Implemented
- Program Counter (PC)
- Instruction Memory
- Instruction Decoder
- Control Unit
- Register File
- Arithmetic Logic Unit (ALU)
- Data Memory
- Processor-Level Integration

## Verification
Each module was verified using dedicated testbenches and waveform analysis in Vivado.

## Tools Used
- Verilog HDL
- Xilinx Vivado
